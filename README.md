# streamdeck-stopwatch

A count-up stopwatch timer plugin for [StreamController](https://github.com/StreamController/StreamController).

## Usage

- **Short press**: Start / Pause / Resume
- **Long press (hold)**: Reset to 00:00

Displays elapsed time with a green progress ring that fills over each 60-second cycle.

## Installation

Clone into your StreamController plugins directory:

```bash
cd ~/.var/app/com.core447.StreamController/data/plugins/
git clone git@github.com:andynu/streamdeck-stopwatch.git local_stopwatch
```

Restart StreamController to load the plugin.
